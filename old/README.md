# オープンレンジブレイクアウト戦略 バックテスト

Refinitiv APIを使用した日本株市場におけるオープンレンジブレイクアウト戦略のバックテストシステム

## 概要

東証プライム主要銘柄を対象に、寄り付き後のレンジブレイクアウトを検出し、イントラデイ取引の収益性を検証するシステムです。

## 戦略ルール

### レンジ計算
- **09:05～09:15（JST）** の10分間の高値・安値を記録

### エントリー
- **09:15～10:00（JST）** にブレイクアウトが発生した場合エントリー
- 上ブレイク → ロング（買い）
- 下ブレイク → ショート（空売り）

### 決済
- **損切り**: エントリー価格から -0.5%（最適化後）
- **利益目標**: +4.0%（最適化後）
- **時間切れ**: 15:00（JST）終値で強制決済

## 主要な実行結果（2025年10月）

### 全体パフォーマンス（37銘柄）
- **総投資額**: 370,000,000円
- **総合リターン**: -1.98%
- **総取引数**: 669回

### セクター別パフォーマンス

| セクター | 平均リターン | 総損益 |
|---------|------------|--------|
| テクノロジー・通信 | +0.99% | +795,682円 ✅ |
| 商社 | -4.77% | -2,386,060円 ❌ |

### トップパフォーマンス銘柄

1. **ソフトバンクG**: +9.12%
2. **レーザーテック**: +6.73%
3. **日立製作所**: +4.22%
